# 🚀 Deval Garg Portfolio Website

A modern **Single Page Application (SPA) Portfolio Website** built using **React.js**, **Material UI**, **React Bootstrap**, and **React Router DOM**.

This portfolio is designed with a clean minimal navbar and an interactive menu navigation system.

---

## 📌 Project Overview

This is a personal portfolio website that includes:

- A centered landing page (Home)
- A navigation hub (Menu Page)
- Dedicated pages for:
  - About Me
  - Projects Section
  - Contact & Social Links

The project focuses on:

✅ Smooth UI  
✅ Responsive layout  
✅ Professional design  
✅ Recruiter-friendly structure  

---

- No page reloads while navigating
